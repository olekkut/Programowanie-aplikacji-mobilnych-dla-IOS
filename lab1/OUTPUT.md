# Rezultat uruchomienia programu (Output) — Lab 1

## Uruchomienie programu
Aby wygenerować poniższy rezultat na własnym komputerze, należy zainstalować Node.js, przejść do katalogu `lab1` i wykonać polecenie:
```bash
node rozwiazania.js
```
